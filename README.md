# FLOPS Technologies - Modern Digital Solutions

![FLOPS Technologies](https://primesparkio.vercel.app/logo.png)

FLOPS Technologies is a premium, high-performance landing page for a modern digital agency. Built with **Next.js 14**, **Tailwind CSS**, and **Framer Motion**, it features a stunning glassmorphism aesthetic, smooth animations, and full responsiveness.

## ğŸ“¦ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/HariDevloper/FLOPS-landingpage.git
   cd FLOPS-landingpage
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file in the root directory and add your configuration (e.g., for email services).

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.
